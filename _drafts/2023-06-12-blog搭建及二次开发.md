1. 先安装jekyll的相关依赖，需要翻墙
   ```bash
    set http_proxy=http://127.0.0.1:7890
    set https_proxy=http://127.0.0.1:7890
    bundle install
   ```
2. 启动服务
   ```bash
   bundle exec jekyll server -H 0.0.0.0ss
   ```
